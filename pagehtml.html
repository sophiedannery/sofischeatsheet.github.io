How to link a CSS file to an HTML file?

Write this code in the header section
n = path to the CSS file

<link href=’n’ rel=’stylesheet’>

How to link a relative path?

Write this code in the header section
n = path to the CSS file

<link href=’n’ rel=’stylesheet’>
